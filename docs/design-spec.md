# 设计说明

## 配色方案
- 主色：#FF2442（红色，小红书标志性颜色）
- 辅助色：#FF5777（浅红色，用于次要操作）
- 强调色：#333333（深灰色，用于重要信息）
- 背景色：#FFFFFF（白色）
- 卡片背景：#F8F8F8（浅灰色）
- 文字次要：#999999（灰色）

## 字体
- 标题：System Default（系统默认字体）
- 正文：System Default（系统默认字体）
- 字号：标题 18-24px，正文 14-16px，辅助文字 12px

## 核心页面
1. **首页** - 笔记信息流，展示用户发布的图文笔记卡片，包含头像、用户名、标题、图片、话题标签、点赞评论收藏按钮
2. **发现页** - 搜索框（支持语音搜索）、热门话题横向滚动、瀑布流图片布局
3. **笔记详情页** - 笔记完整内容、图片轮播、话题标签、评论区（支持楼中楼回复）
4. **发布页** - 图片上传区域（最多9张）、标题输入框、内容文本域、话题选择、发布按钮
5. **通知页** - 消息标签页（全部/点赞/评论/关注）、通知列表
6. **个人主页** - 头像昵称、粉丝关注数、笔记/点赞/收藏tab切换、编辑资料入口

## 交互说明
- 点击底部导航栏切换5个主要Tab
- 点击笔记卡片跳转到笔记详情页
- 点击发布按钮跳转到发布页面
- 下拉刷新首页笔记列表
- 左滑删除评论
- 点击话题标签跳转到话题详情页
- 语音按钮触发语音搜索
- 点击图片进入大图预览模式
